API

APIs can be accessed on the links listed below.

Lists of objects

  • GET Game list: thundergames.pythonanywhere.com/api/games.
  • GET Main genre list: thundergames.pythonanywhere.com/api/main-genres.
  • GET Subgenre list: thundergames.pythonanywhere.com/api/subgenres.

Individual objects

  • GET single Game object: /api/game/<primary_key>, e.g. thundergames.pythonanywhere.com/api/game/1/.
  • GET single Main genre object: /api/main-genre/<primary_key>, e.g. thundergames.pythonanywhere.com/api/main-genre/1/.
  • GET single Subgenre object: /api/subgenre/<primary_key>, e.g. thundergames.pythonanywhere.com/api/subgenre/1/.
